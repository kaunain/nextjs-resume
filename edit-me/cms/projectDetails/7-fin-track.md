**Project description:**
Fin-Track is my flagship open-source engineering project and a production-grade Wealth Management Platform built to demonstrate modern backend engineering and enterprise software architecture.

The platform enables users to manage Stocks, Mutual Funds, ETFs, Gold, NPS, and other investment assets while providing portfolio analytics, market insights, profit & loss tracking, asset allocation, and tax optimization.

Instead of creating another CRUD application, the goal of Fin-Track is to build a scalable enterprise product using Microservices, clean architecture, cloud-native design, and real-world engineering practices.


**Key Contributions**
* Designed complete Microservices architecture from scratch.
* Developing Authentication, User, Portfolio, Market Data, Notification, and Reporting services.
* Building secure REST APIs using Java Spring Boot and JWT Authentication.
* Implementing scalable backend architecture with PostgreSQL, Docker, and API Gateway.
* Following enterprise development practices including documentation, unit testing, CI/CD, GitHub workflow, and production-ready code.
